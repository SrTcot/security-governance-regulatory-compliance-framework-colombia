# security-governance---regulatory-compliance-framework-colombia
Este repositorio consolida un marco integral de seguridad de la información, ciberseguridad, riesgo operativo, continuidad del negocio y cumplimiento normativo aplicable al sector financiero colombiano. Está diseñado como referencia técnica para equipos de ingeniería, ciberseguridad, riesgo, cumplimiento y auditoría.
